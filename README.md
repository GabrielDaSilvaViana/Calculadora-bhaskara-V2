# Calculadora-bhaskara-V2
O que é?
Essa é uma calculadora simples feita para ajudar a resolver equações quadráticas do tipo:

[ ax^2 + bx + c = 0 ]

Com ela, você consegue calcular as raízes dessa equação rapidamente, seja para raízes reais ou complexas.

Funcionalidades
Calcula as raízes da equação quadrática usando a fórmula de Bhaskara.
Gera um gráfico interativo da equação (y = ax^2 + bx + c) no intervalo de (x = -10) a (x = 10).
Se você esquecer de preencher algum coeficiente ou colocar valores errados, a calculadora vai te avisar.
Como usar?
Abra o arquivo index.html no seu navegador.
Preencha os coeficientes (a), (b) e (c) da equação.
Clique em "Calcular Raízes".
O resultado das raízes será mostrado abaixo, junto com o gráfico da equação.
Exemplo:

Para a equação (x^2 - 3x + 2 = 0):

Coloque (a = 1), (b = -3), e (c = 2).
As raízes serão: (x₁ = 2) e (x₂ = 1).
E o gráfico correspondente vai ser gerado logo abaixo.
